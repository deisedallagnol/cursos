Operadores

Os operadores lógicos AND e OR

foo || bar; // retorna 1, que é verdadeiro bar || foo; // retorna 1, que é verdadeiro foo && bar; // retorna 0, que é falso foo && baz; // retorna 2, que é verdadeiro baz && foo; // retorna 1, que é verdadeiro


Operadores de comparação
